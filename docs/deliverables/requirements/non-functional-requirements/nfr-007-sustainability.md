# ID: RDD-NFR-2025-007

# 非機能要件: 持続可能性

## カテゴリ

運用

### 概要

システムが長期的に安定して稼働し、環境負荷を低減しつつ、将来の技術変化やビジネス要件の変化に対応できる能力に関する要件を定義します。これは、資源の効率的な利用と持続可能な成長を両立させるために重要です。

### 詳細

- 最新の安定版技術スタックを採用すること。
- 特定のベンダーに過度に依存しないオープンな技術を採用すること。
- コードは保守性を考慮した設計とコーディング規約に準拠すること。
- ドキュメントは常に最新の状態に保たれること。

### 測定方法

- 技術スタックのバージョン
- ベンダーロックインの度合い
- コードレビューの実施状況と指摘事項
- ドキュメントの更新頻度と正確性
